# 💱 Tarefa 12 - Currency Wallet

> Aplicação mobile desenvolvida em React Native para monitoramento e conversão de moedas internacionais em tempo real.

---

## 📖 Sobre o Projeto

O Currency Wallet foi desenvolvido com o objetivo de consumir dados financeiros em tempo real e apresentar informações sobre diferentes moedas internacionais de forma intuitiva e organizada.

A aplicação consulta uma API de cotações para exibir valores atualizados e simular operações de compra e venda, proporcionando uma experiência semelhante a aplicativos financeiros utilizados no mercado.
